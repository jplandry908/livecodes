## Usage

AsciiDoc lets you write technical docs using plain-text syntax. It's used for books, guides, and software docs.

## Language Info

### Name
AsciiDoc

### Aliases
asciidoc, adoc

### Extensions
.adoc

## Editor

```js
script: {
  language: "asciidoc",
  extension: "adoc"
}
```

## Compiler
AsciiDoctor (via WASM)

### Version
2.x (via AsciiDoctor.js)

## Code Formatting
Not supported directly — formatting is manual.

## Custom Settings

```json
{
  "asciidoc": {
    "attributes": {
      "source-highlighter": "highlight.js",
      "icons": "font"
    }
  }
}
```

## Helper Methods
N/A

## Live Reload
✅ Supported — updates render on change.

## Limitations

- No server-side rendering
- Some advanced extensions may not work (e.g., diagrams)

## Example Usage

```asciidoc
= LiveCodes AsciiDoc Demo

== Features

console.log("Hello from AsciiDoc!")

```
